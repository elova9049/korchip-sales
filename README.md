# 🏭 슈퍼캡 제조 관리 시스템

## 🚀 시작하기
1. **필수 설치**  
   ```bash
   git clone https://github.com/당신의아이디/supercap-system.git
   cd supercap-system
   pip install -r requirements.txt
   ```

2. **데이터베이스 초기화**  
   ```bash
   python database.py
   ```

3. **웹 앱 실행**  
   ```bash
   streamlit run app.py
   ```
